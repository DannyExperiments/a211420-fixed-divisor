WHAT WAS PROVED

For natural n and positive r, define

  a_n = (8n)! n! / ((4n)! (3n)! (2n)!)
  L_r = lcm(1,2,...,8r-1).

For every k in {1,2,3}, the claimed theorem is

  product_{i=1}^r (k*n+i) divides L_r^r * a_n.

The primary Lean source is A211420.lean. It first proves factorial-ratio
integrality and the division-free theorem

  (product_{i=1}^r (k*n+i)) * (4n)! * (3n)! * (2n)!
    divides
  L_r^r * (8n)! * n!,

then proves exact quotient recovery and derives the quotient theorem. It also
proves L_r^r > 0. The constant is exactly L_r^r, not ((8r)!)^r, and C=0 is
not used.

PRIMARY FULLY QUALIFIED LEAN DECLARATIONS

  A211420.integral_cross

  A211420.strengthened_cross

    (hk : k = 1 ∨ k = 2 ∨ k = 3) :
    A211420.D k r n * A211420.ADen n ∣
      A211420.L r ^ r * A211420.ANum n

  A211420.constant_pos
    (r : Nat) (hr : 0 < r) : 0 < A211420.L r ^ r

  A211420.ADen_mul_a
    (n : Nat) :
    A211420.ADen n * A211420.a n = A211420.ANum n

  A211420.strengthened
    (n r k : Nat) (hr : 0 < r)
    (hk : k = 1 ∨ k = 2 ∨ k = 3) :
    A211420.D k r n ∣ A211420.L r ^ r * A211420.a n

Here A211420.L r is Nat.lcmUpto (8*r-1), which mathlib defines as the
finite lcm over the integers from 1 through 8*r-1. A211420.D k r n is the
product over i=1,...,r. A211420.a n is a natural-number quotient only after
integral_cross is proved.
